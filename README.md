
The dataakCrawler is Python web crawler.
It uses the requests and Beautiful Soup (bs4) modules, which help keep the code as simple as possible.

# Getting started

* Install requirements:
````
pip3 install -r requirements.txt
````
* Run the application
````
python3 manage.py 
````
